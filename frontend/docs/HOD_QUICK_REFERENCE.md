## ðŸ“Š Feedback Management Workflow

### Step-by-Step Process:

**1. Navigate to Feedback Management**
   - Click "Feedback Management" from sidebar or dashboard

**2. Select Batch**
   - Choose from B.Tech (4 batches) or M.Tech (2 batches)
   - Batches shown: 2025-2029, 2024-2028, 2023-2027, 2022-2026

**3. Select Regulation**
   - Choose R20 or R23
   - Click regulation button to proceed

**4. Select Semester**
   - Only current year semesters shown
   - Example: Year 3 shows only 3-1 and 3-2
   - Status badges: Published (green) or Draft (yellow)

**5. Review Timetable**
   - View 10 subjects with faculty mapping
   - Subject types: Theory, Lab, Elective, Minor, Honors
   - **Note**: Cannot edit timetable (Admin-managed)

**6. Publish Feedback**
   - Click "Publish Feedback" button
   - Confirm in modal
   - Students can now access feedback form

## ðŸ“ˆ Analytics Workflow

### Step-by-Step Process:

**1. Navigate to Feedback Analytics**
   - Click "Feedback Analytics" from sidebar or dashboard

**2. Select Batch & Regulation**
   - Same as feedback management
   - Batches with data show "Has Data" badge

**3. Select Semester**
   - View response counts and average ratings
   - Color-coded rating badges

**4. View Analytics Report**
   - **Statistics Cards**: Overall avg, excellent parameters, total responses, top subjects
   - **Question-wise Analysis**: 20 questions with bar charts
   - **Subject-wise Performance**: Table with all subjects and faculty ratings

**5. Export Report (Optional)**
   - Click "Export" button
   - Choose PDF or Excel format
   - Download report

**6. Refresh Data**
   - Click "Refresh" button for latest data
   - Analytics update in real-time as students submit

## âš ï¸ Important Notes

1. **Timetable is Read-Only**: HODs cannot modify subject-faculty mappings. Contact Admin for changes.

2. **Feedback is Anonymous**: No way to identify individual students. This ensures honest feedback.

3. **Real-time Analytics**: Analytics update as students submit. Check anytime for latest data.

4. **Branch-Specific Access**: HODs see only their branch's data (e.g., CSE HOD sees only CSE batches).

5. **Semester Visibility**: Only current year semesters shown (past semesters hidden).

6. **Publish Once**: After publishing, feedback form becomes active for students. Cannot unpublish.

## ðŸ› Troubleshooting

### Cannot See Expected Batches:
- Verify you're logged in as correct department HOD
- Check with Admin if batches configured correctly

### Analytics Not Showing:
- Ensure feedback has been published for that semester
- Verify students have submitted responses
- Try refreshing the page

### Export Not Working:
- Check browser's download settings
- Ensure pop-up blocker not blocking download
- Contact support if issue persists

### Password Change Failed:
- Verify current password is correct
- Ensure new password meets all requirements
- Check that passwords match
